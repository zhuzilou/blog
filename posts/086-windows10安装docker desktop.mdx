## 步骤

1. 首先检查电脑是否开启了 wsl

```cmd
wsl
```

![正常开启wsl的样子](http://116.198.244.73:9000/images/20250424233055.png)

- 如果没有开启 wsl，需要从 控制面板->程序->启用或关闭 Windows 功能 中开启`Hyper-v`和`适用于 Linux 的 Windows 子系统`两个选项。
- 如果没有 Hyper-v 选项，可能是系统是家庭版，需要升级为企业/专业版。
- 断开网络，然后找到：设置-》系统-》系统信息》更改产品密钥或升级 Windows，输入密匙重启即可

```plaintext
W269N-WFGWX-YVC9B-4J6C9-T83GX

236TW-X778T-8MV9F-937GT-QVKBB

WMN7B-Y7TKF-Y49QB-TMQ8T-GMT6T

FJHWT-KDGHY-K2384-93CT7-323RC
```

- 重启后再开启 hyper-v 和 wsl，再重启一次

2. 安装 docker desktop
